# 上门服务 - 用户协议与免责条款 (入户服务版)

## 1. 服务性质说明
“安电通”仅作为信息撮合平台，为您链接具有相关资质的维修人员（以下简称“师傅”）。除非特别标注为“平台自营”，否则师傅为独立经营者，非平台员工。

## 2. 入户安全与行为规范
为保障双方权益，请在师傅上门期间遵守以下规范：
2.1 **陪同要求**：师傅作业期间，建议家中留有成年人全程陪同。
2.2 **财产保管**：请您在师傅上门前妥善保管家中贵重物品（如现金、首饰等）。平台不对非因师傅故意行为导致的物品遗失承担责任。
2.3 **隐私区域**：请明确告知师傅作业区域，除作业必要外，不得进入卧室等私密空间。

## 3. 责任限制与免责
3.1 **维修风险**：在排查电路故障过程中，可能会涉及断电测试，由此可能导致的数据丢失（如电脑未保存文件）或设备重启风险，由用户自行承担。
3.2 **不可抗力**：因交通拥堵、恶劣天气等导致师傅无法按预约时间到达的，平台将尽力协调但不承担违约赔偿责任。
3.3 **纠纷处理**：如发生服务质量纠纷或意外财产损失，用户应第一时间向平台投诉。平台将协助取证并按照保险条款（如有）协助理赔，但平台承担的责任上限不超过本次订单服务费用的 3 倍。

## 4. 价格与支付
4.1 **预估报价**：系统显示的“起步价”或“预估价”仅供参考，最终价格由师傅根据现场实际工程量、耗材使用情况与您确认。
4.2 **线上支付**：为保障资金安全，请务必通过平台线上支付服务费。**严禁私下与师傅现金或微信个人转账**，私下交易产生的一切纠纷平台概不负责。
