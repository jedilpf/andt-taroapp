# 社区出行 - 出行服务条款

## 1. 平台定位
本平台“社区出行”板块为聚合打车服务，车辆及驾驶员由合作的第三方网约车平台或具有运营资质的出租车公司提供。

## 2. 乘车安全告知
2.1 **实名乘车**：为保障行程安全，您同意平台收集您的乘车时间、上下车地点及行程录音（部分车辆），并根据监管要求同步给相关政府部门及承运方。
2.2 **紧急联系人**：建议您在 App 中设置紧急联系人，开启“特别是夜间行程自动分享行程”功能。

## 3. 责任界定
3.1 **客运责任**：实际承运方（网约车公司/出租车公司）对运输过程中的安全及服务质量承担主体责任。如发生交通事故或司乘纠纷，本平台将全力协助用户进行维权，但作为信息聚合方，不直接承担承运人责任。
3.2 **物品遗失**：请在下车前检查随身物品。如物品遗落在车上，请立即联系司机或平台客服。平台将协助寻找，但无法保证必定找回。

## 4. 费用与发票
4.1 **预估与实付**：因拥堵、绕路或更改目的地导致的实际车费与预估价不符，以打表或第三方平台计费规则为准。
4.2 **发票开具**：出行服务的电子发票通常由实际承运方（如滴滴、高德等合作方）提供，本平台提供开票入口指引。
