## 代码规范

### TypeScript规范
```typescript
// ✅ 必须定义完整接口
interface InspectionOrder {
  id: number;
  orderNo: string;
  userId: number;
  electricianId?: number;
  status: OrderStatus;
}

// ❌ 禁止使用any
const data: any = response.data;
```

### Java规范
```java
// ✅ 使用MyBatis-Plus
@Mapper
public interface InspectionOrderMapper extends BaseMapper<InspectionOrder> {
}

// ❌ 禁止手写SQL
@Select("SELECT * FROM inspection_order")
List<InspectionOrder> selectAll();
```

## API接口规范

### 用户端接口
| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 检查资格 | GET | /api/inspection/quota | 检查免费检测资格 |
| 创建预约 | POST | /api/inspection/create | 创建检测预约 |
| 订单列表 | GET | /api/inspection/list | 获取订单列表 |
| 订单详情 | GET | /api/inspection/{id} | 获取订单详情 |
| 检测报告 | GET | /api/inspection/report/{id} | 获取报告详情 |
| 创建整改 | POST | /api/inspection/rectify | 创建整改订单 |

### 电工端接口
| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 待抢订单 | GET | /api/electrician/pending | 获取待抢订单 |
| 接单 | POST | /api/electrician/accept/{id} | 接受订单 |
| 到达确认 | POST | /api/electrician/arrive/{id} | 确认到达 |
| 开始检测 | POST | /api/electrician/start/{id} | 开始检测 |
| 提交报告 | POST | /api/electrician/report | 提交检测报告 |

## 数据库规范

### 表命名
- 系统表: `sys_xxx`
- 业务表: `模块_xxx` (如 `inspection_order`)

### 字段命名
- 全部使用小写+下划线
- 主键: `id`
- 创建时间: `create_time`
- 更新时间: `update_time`
- 删除标记: `deleted`

### 索引规范
- 常用查询字段建立索引
- 索引命名: `idx_字段名`
