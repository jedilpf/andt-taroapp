## 代码规范

### 页面规范
```tsx
import Taro from '@tarojs/taro';
import { View, Text } from '@tarojs/components';
import { Button, Input, Toast } from '@nutui/nutui-react-taro';

class InspectionPage extends Taro.Component {
  handleClick = () => {
    Taro.navigateTo({ url: '/pages/user/inspection/book' });
  };

  render() {
    return (
      <View className="inspection-page">
        <Text>检测页面</Text>
        <Button type="primary" onClick={this.handleClick}>
          预约检测
        </Button>
      </View>
    );
  }
}
```

### 组件规范
```tsx
import React from 'react';
import { View, Text } from '@tarojs/components';
import './InspectionCard.scss';

interface InspectionCardProps {
  orderNo: string;
  status: string;
  scheduledTime: string;
  onClick?: () => void;
}

const InspectionCard: React.FC<InspectionCardProps> = ({
  orderNo,
  status,
  scheduledTime,
  onClick
}) => {
  return (
    <View className="inspection-card" onClick={onClick}>
      <View className="card-header">
        <Text className="order-no">{orderNo}</Text>
        <Text className="status">{status}</Text>
      </View>
      <View className="card-body">
        <Text>预约时间: {scheduledTime}</Text>
      </View>
    </View>
  );
};

export default InspectionCard;
```

## 状态管理 (Zustand)

```typescript
import { create } from 'zustand';

interface UserState {
  userInfo: UserInfo | null;
  setUserInfo: (info: UserInfo) => void;
  logout: () => void;
}

export const useUserStore = create<UserState>((set) => ({
  userInfo: null,
  setUserInfo: (info) => set({ userInfo: info }),
  logout: () => set({ userInfo: null }),
}));
```

## 禁止事项

1. ❌ 禁止使用 React + Vite 直接开发
2. ❌ 禁止使用 Ant Design / AntD Mobile
3. ❌ 禁止使用 JavaScript（必须用TypeScript）
4. ❌ 禁止省略TypeScript类型定义
5. ❌ 禁止使用React Router（必须用Taro API）
