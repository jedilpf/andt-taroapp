## 二、软件用途和技术特点

### 2.1 软件用途

安电通是一款专注于家庭用电安全的O2O服务平台软件，主要用途包括：

1. **用户端功能**
   - 在线报修：用户可通过APP或小程序快速提交用电维修需求
   - 订单追踪：实时查看电工位置和服务进度
   - 智能诊断：AI辅助识别用电故障类型
   - 积分商城：消费返积分，积分换商品
   - 评价系统：对电工服务进行评价

2. **电工端功能**
   - 任务大厅：查看并抢单
   - 接单管理：管理已接订单
   - 收入统计：查看收入明细
   - 认证体系：电工资质认证

3. **物业端功能**
   - 小区管理：管理合作小区
   - 订单监控：监控小区内订单
   - 数据报表：查看运营数据
   - 商城管理：管理积分商城

### 2.2 技术特点

1. **前端技术**
   - React + TypeScript + Taro 跨平台框架
   - 支持微信小程序、APP、H多端运行
   - 实时地图定位（高德地图API）
   - WebSocket 实时通信

2. **后端技术**
   - Spring Cloud 微服务架构
   - MySQL + Redis 数据存储
   - 消息队列（RabbitMQ）
   - AI诊断模块

3. **安全特性**
   - 微信/支付宝安全支付
   - 数据加密传输（HTTPS）
   - 用户隐私保护
   - 电工资质审核
