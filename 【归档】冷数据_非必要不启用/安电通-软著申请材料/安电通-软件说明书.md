## 第一章 概述

### 1.1 软件简介

安电通是一款专注于家庭用电安全的O2O服务平台软件，通过连接专业电工与社区居民家庭用户，解决传统电工服务行业"找电工难、价格不透明、服务无保障"的三大民生痛点。

核心理念："像打车一样找电工"——通过实时地图、智能匹配、透明定价、服务保障四大核心能力，打造专业的家庭用电安全服务平台。

### 1.2 功能特点

1. **在线报修**：用户可通过APP或小程序快速提交用电维修需求
2. **智能匹配**：系统自动匹配附近专业电工
3. **实时追踪**：实时查看电工位置和服务进度
4. **透明定价**：199元/299元透明定价，无隐形消费
5. **服务保障**：服务完成后再付款，不满意可投诉
6. **积分商城**：消费返积分，积分换商品

### 1.3 适用对象

- **用户端**：社区居民家庭用户，特别是老年用户、上班族、租房青年
- **电工端**：持证电工师傅，寻求灵活就业机会
- **物业端**：物业公司管理人员，管理小区服务

## 第二章 系统安装与配置

### 2.1 用户端安装

1. 扫描二维码下载APK
2. 允许安装未知来源应用
3. 点击安装

2. 点击"获取"下载
3. 安装完成后打开APP

## 第三章 用户端操作指南

### 3.1 注册与登录

#### 3.1.1 注册流程

1. 打开安电通小程序或APP
2. 点击"我的"页面
3. 点击"登录/注册"
4. 输入手机号，获取验证码
5. 设置密码（可选）
6. 完成注册

#### 3.1.2 登录方式

- 手机号+验证码登录
- 微信一键登录
- 手机号+密码登录

### 3.2 首页功能

#### 3.2.1 服务入口

首页提供以下服务入口：

| 服务类型 | 功能描述 |
|----------|----------|
| 应急维修 | 紧急用电故障维修 |
| 免费检测 | 用电安全免费检测 |
| 便民安装 | 开关插座安装 |
| 智慧家居 | 智能设备安装调试 |

#### 3.2.2 活动Banner

首页轮播展示当前活动：

- 新用户优惠活动
- 节日促销活动
- 安全知识科普

### 3.3 在线报修

#### 3.3.1 提交报修

1. 点击首页"应急维修"
2. 选择故障类型：
   - 跳闸/断电
   - 开关故障
   - 插座故障
   - 灯具故障
   - 线路问题
   - 其他问题
3. 填写故障描述
4. 上传故障照片（最多3张）
5. 选择服务地址
6. 选择服务时间
7. 确认提交

#### 3.3.2 故障类型说明

| 故障类型 | 常见问题 | 建议处理 |
|----------|----------|----------|
| 跳闸/断电 | 空气开关跳闸、全屋断电 | 紧急处理 |
| 开关故障 | 开关失灵、接触不良 | 预约维修 |
| 插座故障 | 插座无电、接触不良 | 预约维修 |
| 灯具故障 | 灯不亮、闪烁 | 预约维修 |
| 线路问题 | 线路老化、短路 | 紧急处理 |

## 第四章 电工端操作指南

### 4.1 注册与认证

#### 4.1.1 注册流程

1. 下载电工端APP
2. 使用手机号注册
3. 设置密码
4. 完善个人信息

#### 4.1.2 资质认证

电工需提交以下材料：

| 材料类型 | 要求 |
|----------|------|
| 身份证 | 正反面照片 |
| 电工证 | 有效期内的电工操作证 |
| 健康证 | 有效期内的健康证 |
| 银行卡 | 用于收款 |

#### 4.1.3 认证审核

- 提交材料后1-3个工作日审核
- 审核通过后可开始接单
- 审核不通过需补充材料

### 4.2 任务大厅

#### 4.2.1 订单列表

订单列表显示：

- 订单类型
- 服务地址
- 距离
- 预估收入
- 服务时间

#### 4.2.2 抢单规则

- 认证电工可抢单
- 同时最多接3单
- 超时未接自动取消
- 抢单成功后需在30分钟内到达

#### 4.2.3 筛选条件

- 按距离筛选
- 按类型筛选
- 按价格筛选
- 按时间筛选

### 4.3 接单管理

#### 4.3.1 进行中订单

1. 查看订单详情
2. 一键导航到用户位置
3. 一键联系用户
4. 开始服务
5. 完成服务
6. 上传服务照片

#### 4.3.2 服务流程

1. 接单后联系用户确认
2. 导航前往服务地点
3. 到达后点击"开始服务"
4. 完成维修后上传照片
5. 用户确认后点击"完成服务"
6. 等待用户支付

### 4.4 收入统计

#### 4.4.1 收入明细

- 今日收入
- 本月收入
- 累计收入
- 订单数量

#### 4.4.2 提现功能

1. 进入"收入"页面
2. 点击"提现"
3. 输入提现金额
4. 确认提现
5. 1-3个工作日到账

#### 4.4.3 收入规则

- 服务费：订单金额的80%
- 平台服务费：订单金额的20%
- 满意度奖励：评分4.8以上额外奖励5%

#### 4.5.1 技能认证

|----------|------|------|

## 第六章 常见问题解答

### 6.1 用户端FAQ

**Q1：如何取消订单？**
A：在订单详情页点击"取消订单"，服务前取消全额退款。

**Q2：支付失败怎么办？**
A：检查网络连接，确认支付密码正确，或更换支付方式。

**Q3：如何投诉电工？**
A：在订单详情页点击"投诉"，填写投诉原因，客服会在24小时内处理。

**Q4：积分如何使用？**
A：进入积分商城，选择商品兑换，积分自动抵扣。

**Q5：如何修改手机号？**
A：在个人中心点击"账号安全"，选择"修改手机号"。

### 6.2 电工端FAQ

**Q1：如何提高接单量？**
A：保持高评分、快速响应、多接单提升等级。

**Q2：提现多久到账？**
A：1-3个工作日到账银行卡。

**Q3：认证审核多久？**
A：1-3个工作日完成审核。

**Q4：如何处理用户投诉？**
A：联系客服说明情况，提供服务证明材料。

### 6.3 物业端FAQ

**Q1：如何添加新小区？**
A：联系商务团队开通新小区权限。

**Q2：分成何时结算？**
A：每月5日结算上月收入。

**Q3：如何查看详细报表？**
A：在数据报表页面导出Excel报表。
