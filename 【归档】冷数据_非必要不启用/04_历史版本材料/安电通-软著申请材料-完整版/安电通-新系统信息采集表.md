## 软件功能信息

**开发目的：**

安电通旨在解决传统家庭用电服务行业"找电工难、价格不透明、服务无保障"的民生痛点，通过O2O平台模式连接专业电工与社区居民，实现家庭用电安全的数字化、智能化服务。

**面向领域 / 行业：** 居民服务业 / 社区物业服务

---

## 软件的主要功能

安电通家庭用电安全服务平台软件主要功能包括：

### 一、用户端功能模块

1. **用户注册与认证**

## 软件的技术特点

### 一、架构特点

1. **微服务架构**
   - 采用Spring Cloud微服务架构
   - 服务拆分：用户服务、订单服务、支付服务、评价服务、积分服务等
   - 服务间通信：RESTful API + RabbitMQ消息队列
   - 负载均衡：Nginx + Ribbon

2. **前后端分离**
   - 后端：Spring Cloud + MyBatis-Plus + MySQL + Redis
   - 前端：React + TypeScript + Taro（多端统一）
   - 移动端：UniApp（编译为小程序、APP）

3. **高可用设计**
   - 服务集群部署
   - 数据库主从复制
   - Redis集群
   - 熔断降级（Hystrix）

### 二、功能特点

1. **实时性**
   - WebSocket实时通信
   - 地图实时定位
   - 订单状态实时推送
   - 即时通讯功能

2. **安全性**
   - 微信/支付宝官方支付接口
   - 数据加密传输（HTTPS）
   - JWT Token认证
   - 敏感数据脱敏

3. **智能性**
   - AI故障诊断辅助
   - 智能派单算法
   - LBS位置服务
   - 推荐算法

4. **扩展性**
   - 插件化设计
   - 第三方接口集成（地图、支付、短信）
   - 多租户支持

### 三、数据处理特点

1. **数据存储**
   - 关系型数据：MySQL
   - 缓存数据：Redis
   - 消息队列：RabbitMQ
   - 文件存储：七牛云OSS

2. **数据安全**
   - 数据库定期备份
   - 数据加密存储
   - 操作日志记录
   - 权限控制
