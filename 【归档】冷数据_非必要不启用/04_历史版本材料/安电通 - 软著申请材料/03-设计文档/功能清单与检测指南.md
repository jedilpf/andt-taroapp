## 用户端功能

### 1. 核心服务功能 ⚡

#### 1.1 电路急修 (RepairPage)
- **页面路径**: `/user/repair`
- **文件位置**: `pages/user/UserFunctions.tsx`
- **核心按钮**:
  - ✅ "智能分析" - AI诊断故障
  - ✅ "呼叫电工" - 一键下单
  - ✅ "紧急故障" - 快速通道
  - ✅ "常规维修" - 普通预约
  
**调用链路**: 
1. 用户描述故障 → 
2. 调用Gemini AI分析 → 
3. 显示诊断建议 → 
4. 创建急修订单 → 
5. 跳转订单列表

**链路文档**: `功能调用链路.md` - 第7节

---

#### 1.2 深度体检 (DeepInspectionPage)
- **页面路径**: `/user/inspection`
- **文件位置**: `pages/user/services/DeepInspection.tsx`
- **核心按钮**:
  - ✅ "立即预约体检" - 创建检测订单

**服务内容**:
- 配电箱深度检测
- 开关插座排查
- 线路负载与发热
- 绝缘电阻测试

**调用链路**:
用户点击预约 → createOrder(type:'Inspection') → navigate('/user/orders')

**链路文档**: `功能调用链路.md` - 第8节

---

#### 1.3 上门安装 (InstallPage)
- **页面路径**: `/user/install`
- **文件位置**: `pages/user/UserFunctions.tsx`
- **核心按钮**:
  - ✅ 服务类型选择（4种）
  - ✅ 材料加减按钮（+/-）
  - ✅ "立即预约" - 创建安装订单

**服务类型**:
- 灯具安装 (¥30起)
- 开关插座 (¥15起)
- 家电接线 (¥50起)
- 其他安装 (¥30起)

**特色功能**:
- ✅ 自营辅材选购
- ✅ 实时价格计算
- ✅ 材料数量管理

**调用链路**:
选择服务 → 选购材料 → 填写备注 → 创建订单（含材料清单）

**链路文档**: `功能调用链路.md` - 第9节

---

#### 1.4 公益活动 (ActivityPage)
- **页面路径**: `/user/activity`
- **文件位置**: `pages/user/UserFunctions.tsx`
- **核心按钮**:
  - ✅ "立即预约" - 打开Modal
  - ✅ "选择时间" - TimeSelector
  - ✅ "选择地址" - AddressSelector
  - ✅ "确认免费预约" - 创建免费订单

**调用链路**:
点击预约 → Modal弹出 → 选择时间地址 → 创建免费检测订单(¥0)

**链路文档**: `功能调用链路.md` - 第10节

import { test, expect } from '@playwright/test';

test.describe('安电通功能全量检测', () => {

      // 添加材料

      // 预约

      await page.click('text=明天下午');
