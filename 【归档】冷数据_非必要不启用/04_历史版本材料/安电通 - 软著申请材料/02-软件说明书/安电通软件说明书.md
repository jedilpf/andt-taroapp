## 1. 软件概述

### 1.1 软件简介

安电通上门电工服务系统是一款基于移动互联网技术的 O2O（Online to Offline）电工服务平台，旨在解决家庭电路维修需求与专业电工服务之间的信息不对称问题。系统通过 AI 智能诊断和实时地图匹配，实现快速响应、透明消费、全程可追溯的上门电工服务。

### 1.2 软件特点

- **AI 智能诊断**: 基于 DeepSeek/通义千问大模型的电路故障智能分析
- **地图找人**: 实时查看附近电工位置，一键预约
- **双端协同**: 用户端 + 电工端，数据实时同步
- **状态追踪**: 8 状态订单生命周期管理，全程透明
- **公益模式**: 免费上门检测，关爱特殊群体

### 1.3 功能架构

```
安电通系统
├── 用户端
│   ├── AI 智能诊断
│   ├── 电路急修
│   ├── 深度体检
│   ├── 上门安装
│   ├── 订单管理
│   ├── 个人中心
│   ├── 社区互动
│   └── 积分商城
│
├── 电工端
│   ├── 任务大厅
│   ├── 我的任务
│   ├── 收益管理
│   ├── 技能管理
│   └── 服务范围
│
└── AI 服务
    ├── 电路故障诊断
    ├── AI 聊天机器人
    └── 多模型支持
```

## 3. 安装与部署

### 3.1 开发环境部署

#### 步骤 1: 克隆项目

```bash
git clone <项目地址>
cd 安电通
```

#### 步骤 2: 安装依赖

```bash
npm install
```

#### 步骤 3: 配置环境变量

创建 `.env.local` 文件：

```bash
# AI 服务提供商配置
AI_PROVIDER=deepseek

# AI API 密钥
AI_API_KEY=你的_API_KEY
```

#### 步骤 4: 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:5173/ 即可使用。

### 3.2 生产环境部署

#### 步骤 1: 构建生产版本

```bash
npm run build
```

#### 步骤 2: 预览构建结果

```bash
npm run preview
```

#### 步骤 3: 打包为移动端应用

```bash
# 同步 Capacitor
npx cap sync

# 添加 Android 平台
npx cap add android

# 打开 Android Studio
npx cap open android
```

在 Android Studio 中构建 APK 文件。

### 3.3 服务器部署（可选）

将 `dist` 目录部署到任意静态 web 服务器：

**Nginx 配置示例**:

```nginx
server {
    listen 80;
    server_name andiantong.example.com;
    root /var/www/andiantong/dist;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

## 4. 功能说明

### 4.1 用户端功能

#### 4.1.1 登录/注册

- 支持手机号快速登录
- 支持账号密码登录
- 自动记住登录状态

**入口**: 启动应用 → 选择"用户端" → 登录

#### 4.1.2 AI 智能诊断

**功能描述**: 用户描述电路故障现象，AI 提供专业诊断建议。

**操作流程**:
1. 进入"电路急修"页面
2. 在文本框输入故障描述（如"厨房插座冒火花"）
3. 点击"智能分析"按钮
4. 查看 AI 返回的诊断结果
5. 可一键下单预约电工

**技术特点**:
- 支持 DeepSeek、通义千问等多模型
- 本地知识库 fallback，离线也能用
- 3 步建议：安全警告、可能原因、服务类型

#### 4.1.3 电路急修

**功能描述**: 提供家庭电路紧急维修服务。

**服务流程**:
1. 选择故障类型
2. 填写故障描述
3. 选择预约时间
4. 选择服务地址
5. 提交订单
6. 等待电工接单

**收费标准**:
- 上门费：30-50 元
- 检修费：50 元起（视难度而定）
- 材料费：实报实销

#### 4.1.4 深度体检

**功能描述**: 提供全屋电路深度检测服务。

**检测内容**:
- 配电箱安全检测
- 插座开关检测
- 线路老化检测
- 漏电保护检测

**服务流程**:
1. 预约检测时间
2. 电工上门检测
3. 生成检测报告
4. 提供维修建议

#### 4.1.5 上门安装

**功能描述**: 提供灯具、开关、浴霸等安装服务。

**服务类型**:
- 灯具安装（30 元起）
- 开关插座安装（20 元/个）
- 浴霸安装（80 元起）
- 晾衣架安装（100 元起）

**特色功能**:
- 材料选购：在线购买安装材料
- 实时计价：自动计算总费用
- 时间选择：灵活预约上门时间

#### 4.1.6 订单管理

**功能描述**: 查看所有订单状态，追踪服务进度。

**订单状态**:
- 待接单（PENDING）
- 已接单（ACCEPTED）
- 已到达（ARRIVED）
- 施工中（IN_PROGRESS）
- 已完成（COMPLETED）

**功能列表**:
- 订单列表：按时间排序
- 订单详情：查看完整信息
- 联系电工：电话/在线聊天
- 取消订单：符合条件可取消
- 评价服务：服务完成后评价

#### 4.1.7 个人中心

**功能模块**:
- 个人资料：头像、昵称、手机号
- 地址管理：添加/编辑/删除地址
- 优惠券：领取/使用/查看
- 积分管理：积分查询/兑换
- 会员中心：等级权益/购买会员
- 实名认证：身份证上传/审核
- 我的收藏：收藏的电工/服务
- 浏览历史：最近浏览记录
- 设置：消息通知/隐私设置

#### 4.1.8 社区互动

**功能描述**: 用电知识科普、用户案例分享。

**功能列表**:
- 视频流：电工知识短视频
- 点赞评论：互动功能
- 关注系统：关注喜欢的电工
- 知识分类：安全用电、故障排查等

#### 4.1.9 积分商城

**功能描述**: 积分兑换商品和服务。

**兑换类型**:
- 实物商品：小家电、工具等
- 优惠券：满减券、折扣券
- 服务权益：免费上门、优先派单

**积分获取**:
- 每日签到
- 完成订单
- 邀请好友
- 社区互动

## 7. 常见问题

### 7.1 用户端常见问题

**Q1: 如何取消订单？**

A: 进入"我的订单" → 选择订单 → 点击"取消订单"。如电工已接单，需与电工协商取消。

**Q2: 对服务不满意怎么办？**

A: 可在订单完成后进行差评，并联系客服投诉。我们会核实处理。

**Q3: 积分如何获取？**

A: 每日签到、完成订单、邀请好友、社区互动均可获得积分。

**Q4: AI 诊断准确吗？**

A: AI 诊断基于大模型和本地知识库，可提供初步判断。但复杂问题仍需电工上门检测。

### 7.2 电工端常见问题

**Q1: 如何成为电工？**

A: 下载电工端 → 注册账号 → 上传身份证和电工证 → 等待审核 → 审核通过后可接单。

**Q2: 订单如何分配？**

A: 采用抢单模式，电工可在任务大厅自由抢单。

**Q3: 收益如何结算？**

A: 订单完成后，收益自动到账。可申请提现到微信/支付宝。

**Q4: 如何设置服务范围？**

A: 进入"服务范围" → 在地图上选择中心点 → 设置服务半径 → 保存。

### 7.3 技术常见问题

**Q1: 应用无法启动？**

A: 检查网络连接，清除缓存后重试。如仍无法解决，重新安装应用。

**Q2: AI 服务无法使用？**

A: 检查 API Key 配置是否正确，网络连接是否正常。可切换到本地模式。

**Q3: 定位不准确？**

A: 检查 GPS 权限是否开启，尝试重新授权。
