## 🎯 软著申请流程

### 第一步：材料准备（当前阶段）
- ✅ 源代码整理完成
- ✅ 软件说明书编写完成
- ✅ 设计文档整理完成
- ⏳ 功能截图待准备
- ⏳ 申请表待填写

### 第二步：在线填报
1. 访问中国版权保护中心官网：http://www.ccopyright.com.cn/
2. 注册/登录账号
3. 进入"著作权登记" → "计算机软件著作权登记"
4. 在线填写申请表
5. 上传电子版材料

### 第三步：提交纸质材料
1. 打印申请表（盖章/签字）
2. 准备其他纸质材料
3. 邮寄或现场提交到版权保护中心

### 第四步：等待审批
- **审批周期**: 30-45 个工作日
- **补正通知**: 如有问题会通知补正
- **缴费通知**: 审批通过后缴费（约 300 元）

### 第五步：领取证书
- **电子证书**: 官网下载
- **纸质证书**: 邮寄或自取
