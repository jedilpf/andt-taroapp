## 🛠️ 截图技巧

### 方法 1: 浏览器自带截图

**Chrome/Edge**:
1. 按 `F12` 打开开发者工具
2. 按 `Ctrl+Shift+P` 打开命令面板
3. 输入 `screenshot`
4. 选择：
   - `Capture full size screenshot` - 截取完整页面
   - `Capture screenshot` - 截取可见区域
   - `Capture node screenshot` - 截取指定元素

### 方法 2: 截图工具

**推荐工具**:
- **Snipaste** (Windows): 免费、轻量、好用
- **ShareX** (Windows): 功能强大、开源免费
- **系统自带**: Win+Shift+S

### 方法 3: 开发者工具

**移动模式截图**:
1. 切换到移动模式（Ctrl+Shift+M）
2. 点击右上角"..."菜单
3. 选择"Capture screenshot"

## 🆘 常见问题

### Q1: 开发服务器未运行怎么办？

**A**: 运行以下命令启动：
```bash
npm run dev
```

### Q2: 页面显示不正常怎么办？

**A**: 
1. 清除浏览器缓存（Ctrl+Shift+Delete）
2. 刷新页面（Ctrl+R）
3. 检查控制台是否有错误

### Q3: 移动端模式显示不完整怎么办？

**A**: 
1. 切换设备型号（选择更大的设备）
2. 调整缩放比例
3. 使用"Capture full size screenshot"

### Q4: 某些功能无法使用怎么办？

**A**: 
1. 检查是否已登录
2. 检查网络连接
3. 查看控制台错误信息

### Q5: 截图文件太大怎么办？

**A**: 
1. 使用图片压缩工具
2. 降低截图分辨率
3. 转换为 JPG 格式
