## 🎯 核心功能亮点

### 1. AI 智能诊断系统
- **技术**: DeepSeek/通义千问大模型
- **功能**: 电路故障智能分析
- **代码位置**: `services/geminiService.ts`
- **调用链路**: 用户输入 → AI 分析 → 3 步建议 → 一键下单

### 2. 地图找人功能
- **技术**: 高德/百度地图 API
- **功能**: 实时查看附近电工位置
- **代码位置**: `pages/MapPage.tsx`
- **交互**: 底层地图 + UI 分层设计

### 3. 订单状态机
- **状态**: PENDING → ACCEPTED → ARRIVED → IN_PROGRESS → COMPLETED
- **代码位置**: `context/AppContext.tsx`
- **特性**: 8 状态流转、WebSocket 推送

### 4. 双端协同架构
- **用户端**: 下单、支付、评价
- **电工端**: 抢单、施工、收益
- **数据同步**: 实时同步订单状态

## 📝 软著申请流程

### 步骤 1: 准备材料（当前阶段）
- ✅ 源代码（157 个文件）
- ✅ 软件说明书（15000 字）
- ✅ 设计文档（5 份）
- ⏳ 功能截图（待截图）
- ⏳ 申请表（待填写）

### 步骤 2: 在线填报
1. 访问：中国版权保护中心官网
2. 注册/登录账号
3. 填写申请表
4. 上传材料

### 步骤 3: 提交纸质材料
1. 打印申请表
2. 盖章/签字
3. 邮寄或现场提交

### 步骤 4: 等待审批
- **审批周期**: 30-45 个工作日
- **补正通知**: 如有问题会通知补正
- **缴费通知**: 审批通过后缴费

### 步骤 5: 领取证书
- **电子证书**: 官网下载
- **纸质证书**: 邮寄或自取
