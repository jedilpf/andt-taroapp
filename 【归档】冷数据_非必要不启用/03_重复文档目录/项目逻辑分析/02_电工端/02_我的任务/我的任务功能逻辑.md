## 五、电工操作流

### 5.1 到达客户地址

```
电工到达客户地址
       点击"已到├── 更新订单状态为"已到└── 显示"开始作按钮
```

### 5.2 开始作```
点击"开始作├── 更新订单状态为"施工└── 计时开始（可选）
```

### 5.3 完成服务

```
服务完成 点击"服务完成"
       ├── 更新订单状态为"已完├── 通知客户确认
       └── 跳转订单列表
```
