### 10.1 党员电工优先推荐方案

**问题ID**: Q001  
**问题**: 如何在电工列表中优先展示党员电工**解决方案**:

#### Phase-1方案（简单排序）
- 电工列表按身份排序：党员 > 认证 > 普- 党员电工添加"先锋模范"标签

#### Phase-2方案（智能推荐）

**推荐算法**:
```typescript
interface ElectricianScore {
  id: string;
  baseScore: number;
  identityBonus: number;  // 身份加成
  ratingBonus: number;    // 评分加成
  distanceBonus: number;  // 距离加成
  finalScore: number;
}

const calculateScore = (elec: Electrician, userLocation: Location): number => {
  let score = 0;
  
  // 身份加成
  if (elec.identity === 'PARTY') score += 30;
  else if (elec.identity === 'VERIFIED') score += 20;
  
  // 评分加成
  score += elec.rating * 10;
  
  // 距离加成（越近越高）
  const distance = calculateDistance(elec.location, userLocation);
  if (distance < 3) score += 20;
  else if (distance < 5) score += 10;
  
  return score;
};

// 排序逻辑
const sortedElectricians = electricians
  .map(elec => ({
    ...elec,
    score: calculateScore(elec, userLocation)
  }))
  .sort((a, b) => b.score - a.score);
```

- 党员电工显示红色边框
- 列表顶部显示"党员先锋"专区
- 添加"优先推荐"标签

### 10.2 服务类型定价方案

**问题ID**: Q002  
**问题**: 不同服务类型如何定价**解决方案**:

#### Phase-1方案（统一价格- 所有服务类型使用电工起步价
- 实际费用上门后协#### Phase-2方案（分类定价）

**定价规则**:
```typescript
const servicePricing = {
  Repair: {
    name: '电路急修',
    basePrice: 40,
    unit: ',
    description: '含故障排查、简单维},
  Install: {
    name: '上门安装',
    basePrice: 50,
    unit: '点位',
    description: '含开关、插座、灯具安},
  Checkup: {
    name: '线路检,
    basePrice: 80,
    unit: ',
    description: '含全屋电路安全检}
};

// 价格计算
const calculatePrice = (type: string, electrician: Electrician) => {
  const basePrice = servicePricing[type].basePrice;
  const elecMinPrice = parseInt(electrician.priceStr.replace(/\D/g, ''));
  return Math.max(basePrice, elecMinPrice);
};
```

**UI展示**:
- 服务类型卡片显示价格
- 价格区间动态计- 显示服务包含内容
