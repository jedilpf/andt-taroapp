## 十、待确认问题及解决方

### 10.1 检测报告生成方**问题ID**: Q001  
**问题**: 检测完成后如何生成专业检测报告？  
**解决方案**:

#### Phase-1方案（简化报告）
- 电工手动填写检测结- 系统生成标准化报告模- PDF格式下载

#### Phase-2方案（智能报告）

**报告内容结构**:
```typescript
interface InspectionReport {
  orderId: string;
  inspectionDate: number;
  electricianId: string;
  electricianName: string;
  items: InspectionItem[];
  overallScore: number;  // 安全评分 0-100
  risks: RiskItem[];     // 风险项列recommendations: string[];  // 改善建议
  images: string[];      // 检测照}

interface InspectionItem {
  name: string;
  status: 'PASS' | 'WARNING' | 'FAIL';
  score: number;
  description: string;
  images: string[];
}

interface RiskItem {
  level: 'HIGH' | 'MEDIUM' | 'LOW';
  description: string;
  suggestion: string;
}
```

**报告生成流程**:
1. 电工完成检填写检测数2. 系统自动计算安全评分
3. AI分析风险等级
4. 生成改善建议
5. 用户查看PDF报告

### 10.2 定期检测提醒方**问题ID**: Q002  
**问题**: 如何实现每年一次的定期检测提醒？  
**解决方案**:

#### Phase-1方案（手动提醒）
- 用户在个人中心设置提- 系统在到期前7天推送通知

#### Phase-2方案（智能提醒）

**提醒规则**:
```typescript
interface InspectionReminder {
  userId: string;
  lastInspectionDate: number;
  nextInspectionDate: number;
  reminderDays: number[];  // [7, 3, 1] 提前几天提醒
  enabled: boolean;
}

// 提醒触发逻辑
const checkReminder = () => {
  const daysUntilNext = Math.ceil((nextInspectionDate - Date.now()) / (1000 * 60 * 60 * 24));
  if (reminderDays.includes(daysUntilNext)) {
    sendNotification({
      title: '电路检测提,
      body: `距离下次检测还{daysUntilNext}天，建议提前预约`,
      action: '/user/deep-inspection'
    });
  }
};
```

**推送渠*:
- 小程序订阅消- 短信通知
- App推送通知
