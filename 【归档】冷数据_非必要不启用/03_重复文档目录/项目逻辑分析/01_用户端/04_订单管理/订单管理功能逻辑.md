### 9.1 待付款Tab实现方案

**问题ID**: Q001  
**问题**: 待付款Tab是否Phase-1实现**解决方案**:

#### Phase-1方案（暂不实现）
- 待付款Tab显示但置灰，点击提示"暂未开- 所有订单创建后直接进入"待接状- 不支持货到付款，所有服务先服务后支**选择理由**:
- Phase-1核心目标是快速上线，简化支付流- 电力服务场景特殊：先服务后付费更合理
- 减少支付相关的开发工作量
- 降低用户决策门槛，提高转化率

#### Phase-2方案（完整支付流程）

const schedulePaymentTimeout = (orderId: string, timeout: number) => {

```typescript
interface OrderReview {
  id: string;
  orderId: string;
  userId: string;
  electricianId: string;
  
  overallRating: number;  // 1-5dimensionRatings: {
    attitude: number;     // 服务态度
    professional: number; // 专业水平
    timeliness: number;   // 服务时效
    price: number;        // 价格合理
  };
  
  content: string;       // 文字评价
  images: string[];      // 图片URL
  tags: string[];        // 快捷标签
  
  isAnonymous: boolean;   // 是否匿名
  isVisible: boolean;     // 是否可见
  
  pointsAwarded: number;  // 获得积分
  createdAt: Date;
}

const submitReview = async (review: CreateReviewRequest) => {
  // 1. 验证订单状const order = await getOrder(review.orderId);
  if (order.status !== OrderStatus.COMPLETED) {
    throw new Error('订单未完成，无法评价');
  }
  
  // 2. 检查是否已评价
  const existingReview = await getReviewByOrderId(review.orderId);
  if (existingReview) {
    throw new Error('订单已评);
  }
  
  // 3. 计算积分奖励
  let pointsAwarded = 5; // 基础积分
  
  if (review.content && review.content.length >= 10) {
    pointsAwarded += 15; // 图文评价
  }
  
  if (review.images && review.images.length > 0) {
    pointsAwarded += 5; // 有图}
  
  // 4. 创建评价
  const newReview = await createReview({
    ...review,
    pointsAwarded,
    createdAt: new Date()
  });
  
  // 5. 发放积分
  await awardPoints(review.userId, pointsAwarded, 'REVIEW');
  
  // 6. 更新电工评分
  await updateElectricianRating(review.electricianId);
  
  // 7. 更新订单状await updateOrder(review.orderId, { reviewed: true });
  
  return newReview;
};

const updateElectricianRating = async (electricianId: string) => {
  const reviews = await getElectricianReviews(electricianId);
  
  // 计算平均分（加权const avgRating = reviews.reduce((sum, r) => sum + r.overallRating, 0) / reviews.length;
  
  // 计算各维度平均分
  const dimensionAvg = {
    attitude: reviews.reduce((sum, r) => sum + (r.dimensionRatings.attitude | 0), 0) / reviews.length,
    professional: reviews.reduce((sum, r) => sum + (r.dimensionRatings.professional | 0), 0) / reviews.length,
    timeliness: reviews.reduce((sum, r) => sum + (r.dimensionRatings.timeliness | 0), 0) / reviews.length,
    price: reviews.reduce((sum, r) => sum + (r.dimensionRatings.price | 0), 0) / reviews.length
  };
  
  // 更新电工信息
  await updateElectrician(electricianId, {
    rating: avgRating,
    reviewCount: reviews.length,
    dimensionRatings: dimensionAvg
  });
};
```

### 9.3 删除订单实现方案

**问题ID**: Q003  
**问题**: 删除订单是否真正从数据库删除**解决方案**:

#### 最终方案：软删除（本地隐藏**删除策略**:

| 删除类型 | 数据处理 | 说明 
| 属性 | 接口概览 | 接口名称 |---------| 属性 | 接口概览 | 接口名称 |---------| 属性 | 接口概览 | 接口名称 |------
| 用户删除 | 软删除（标记hidden| 用户端不显示，数据保
| 系统清理 | 定期归档 | 180天后归档到历史库 
| 真实删除 | 物理删除 | 仅在用户注销账号

**选择理由**:
1. **数据完整*：订单数据涉及财务、积分、服务记录，不能随意删除
2. **审计需*：需要保留完整交易记录用于财务审3. **纠纷处理**：用户可能需要查看历史订单解决纠4. **数据分析**：历史订单数据用于业务分析和优化
5. **法律合规**：电商法要求保留交易记录至少3#### 技术实**数据模型**:

```typescript
const deleteOrder = async (
```

**订单归档逻辑**:
```typescript
const archiveOldOrders = async () => {
  // 每天凌晨执行
  const archiveThreshold = new Date();
  archiveThreshold.setDate(archiveThreshold.getDate() - 180); // 180天前
  
  const oldOrders = await queryOrders({
    status: { $in: [OrderStatus.COMPLETED, OrderStatus.CANCELLED] },
    createdAt: { $lt: archiveThreshold },
    isArchived: false
  });
  
  for (const order of oldOrders) {
    // 1. 复制到归档库
    await archiveOrder(order);
    
    // 2. 标记为已归档
    await updateOrder(order.id, {
      isArchived: true,
      archivedAt: new Date()
    });
    
    // 3. 删除主库中的详细数据（保留摘要）
    await minimizeOrderData(order.id);
  }
};

const minimizeOrderData = async (orderId: string) => {
  // 归档后只保留摘要信息
  await updateOrder(orderId, {
    items: null,           // 删除详细商品列表
```

**恢复删除功能**:
```typescript
const restoreOrder = async (orderId: string, userId: string) => {
  // 用户可恢0天内删除的订const order = await getOrder(orderId);
  
  if (order.clientId !== userId) {
    throw new Error('无权操作');
  }
  
  if (!order.isHidden) {
    throw new Error('订单未删);
  }
  
  const hiddenDays = Math.floor(
    (Date.now() - order.hiddenAt!.getTime()) / (1000 * 60 * 60 * 24)
  );
  
  if (hiddenDays > 30) {
    throw new Error('删除超过30天，无法恢复');
  }
  
  await updateOrder(orderId, {
    isHidden: false,
    hiddenAt: null,
    hiddenBy: null
  });
  
  await logOperation({
    type: 'ORDER_RESTORE',
    orderId,
    userId,
    timestamp: new Date()
  });
};
```
