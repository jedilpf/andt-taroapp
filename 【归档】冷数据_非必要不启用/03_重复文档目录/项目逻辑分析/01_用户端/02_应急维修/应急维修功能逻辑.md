### 13.1 派单规则方案

**问题ID**: Q001  
**问题**: 派单规则km内派单还是全城抢单？  
**解决方案**:

#### Phase-1方案km内派单）

- 提高电工工作效率
- 减少电工空跑成本

#### Phase-2方案（智能派抢单混合**混合派单策略**:

| 场景 | 派单方式 | 说明 
| 属性 | 接口概览 | 接口名称 |------| 属性 | 接口概览 | 接口名称 |---------| 属性 | 接口概览 | 接口名称 |------
| 紧急故| 系统派单 | 3km内最优电工，保证响应速度 
| 普通故| 抢单模式 | 5km内电工可抢单，提高效
| 高峰| 混合模式 | 先派单，5分钟无人接单转抢

**派单算法**:
```typescript
interface DispatchStrategy {
  orderId: string;
  faultType: 'URGENT' | 'NORMAL';
  userLocation: Location;
  
  // 派单流程
  // 1. 紧急故3km内系统派// 2. 普通故5km内抢// 3. 5分钟无人接单 扩大范围或转抢单
}

const dispatchOrder = async (order: RepairOrder) => {
  const electricians = await getAvailableElectricians(
    order.location,
    order.urgent 3000 : 5000  // 3kmkm
  );
  
  if (order.urgent) {
    // 紧急故障：系统派单给最优电const best = selectBestElectrician(electricians);
    await assignOrder(order.id, best.id);
  } else {
    // 普通故障：推送给区域内所有电工抢await broadcastToElectricians(order, electricians);
  }
};
```

```typescript
interface PricingFactors {
  baseFee: number;           // 基础费用0distanceFee: number;       // 距离费用：每km +5urgencyFee: number;        // 紧急费用：紧急故+20timeFee: number;           // 时段费用：夜节假+30%
  complexityFee: number;     // 复杂度费用：根据故障类型
}

const calculatePrice = (order: RepairOrder): PriceRange => {
  const factors = analyzePricingFactors(order);
  
  const minPrice = factors.baseFee 
    + factors.distanceFee 
    + factors.complexityFee;
    
  const maxPrice = minPrice 
    + factors.urgencyFee 
    + factors.timeFee;
    
  return { min: minPrice, max: maxPrice };
};

### 13.3 紧急故障派单优先级方案

**问题ID**: Q003  
**问题**: 紧急故障派单优先级具体逻辑**解决方案**:

#### Phase-1方案（简单优先级- 紧急故障标记为P0优先- 系统优先派单给距离最近的电工
- 电工端显紧标识

#### Phase-2方案（加权算法）

**优先级计算公*:
```
优先级得= 距离权重(40%) + 评分权重(30%) + 在线时长权重(20%) + 特殊标识权重(10%)

距离权重 = (3 - distance) / 3 × 40
评分权重 = (rating - 4.0) / 1.0 × 30
在线时长权重 = min(onlineHours / 8, 1) × 20
特殊标识权重 = 党员(10) + 金牌(8) + 优5)
```

const calculatePriority = (
  electrician: Electrician, 
  order: RepairOrder
): number => {
  const distance = calculateDistance(order.location, electrician.location);
  
  const distanceScore = Math.max(0, (3 - distance) / 3) * 40;
  const ratingScore = Math.max(0, (electrician.rating - 4.0) / 1.0) * 30;
  const onlineScore = Math.min(electrician.onlineHours / 8, 1) * 20;
  
  let badgeScore = 0;
  if (electrician.tags.includes('党员')) badgeScore += 10;
  if (electrician.tags.includes('金牌电工')) badgeScore += 8;
  if (electrician.tags.includes('优)) badgeScore += 5;
  
  return distanceScore + ratingScore + onlineScore + badgeScore;
};

const dispatchUrgentOrder = async (order: RepairOrder) => {
  // 1. 获取3km内所有在线电const electricians = await getElectricians(order.location, 3000, 'ONLINE');
  
  // 2. 计算优先级得const ranked = electricians
    .map(e => ({ ...e, priority: calculatePriority(e, order) }))
    .sort((a, b) => b.priority - a.priority);
  
  // 3. 派单给得分最高的电工
  if (ranked.length > 0) {
    await assignOrder(order.id, ranked[0].id);
    
    // 4. 设置超时重派5秒）
    setTimeout(() => retryDispatch(order.id), 15000);
  }
};
```

interface WarrantyClaim {
  orderId: string;
  userId: string;
  claimReason: string;
  photos: string[];  // 故障照片
}

interface WarrantyValidation {
  isValid: boolean;
  warrantyType: string;
  expireDate: Date;
  remainingDays: number;
  exclusions: string[];
}

const validateWarranty = async (orderId: string): Promise<WarrantyValidation> => {
  const order = await getOrder(orderId);
  const warranty = await getWarranty(orderId);
  
  const now = new Date();
  const expireDate = new Date(warranty.startDate);
  expireDate.setDate(expireDate.getDate() + warranty.days);
  
  const remainingDays = Math.ceil((expireDate.getTime() - now.getTime()) / (1000 * 60 * 60 * 24));
  
  return {
    isValid: remainingDays > 0,
    warrantyType: warranty.type,
    expireDate,
    remainingDays,
    exclusions: warranty.exclusions
  };
};

const processWarrantyClaim = async (claim: WarrantyClaim) => {
  // 1. 验证质保
  const validation = await validateWarranty(claim.orderId);
  
  if (!validation.isValid) {
    throw new Error('质保已过);
  }
  
  // 2. 创建质保工单
  const warrantyOrder = await createWarrantyOrder({
    originalOrderId: claim.orderId,
    userId: claim.userId,
    type: 'WARRANTY',
    description: claim.claimReason,
    photos: claim.photos
  });
  
  // 3. 派单给原电工（优先）或其他电await dispatchWarrantyOrder(warrantyOrder);
  
  return warrantyOrder;
};
