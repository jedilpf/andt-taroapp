## 八、API接口

### 8.1 内容接口

```
# 获取学习强国内容
GET /api/community/study/videos
参数:
  - subCategory: 子分类ID (可- page: 页码
  - pageSize: 每页数量

# 获取专题学习
GET /api/community/study/topics
参数:
  - status: 专题状(可# 获取专题详情
GET /api/community/study/topics/:id

# 更新学习进度
POST /api/community/study/progress
参数:
  - userId: 用户ID
  - contentId: 内容ID
  - duration: 学习时长
```

### 8.2 答题接口

```
# 获取每日答题
GET /api/community/study/quiz/daily

# 获取挑战答题
GET /api/community/study/quiz/challenge

# 提交答案
POST /api/community/study/quiz/answer
参数:
  - questionId: 题目ID
  - answer: 答案

# 获取答题记录
GET /api/community/study/quiz/records
参数:
  - userId: 用户ID
  - type: 答题类型
```

### 8.3 积分接口

```
# 获取学习积分
GET /api/community/study/points
参数:
  - userId: 用户ID

# 获取积分记录
GET /api/community/study/points/history
参数:
  - userId: 用户ID
  - page: 页码

# 获取学习排名
GET /api/community/study/ranking
参数:
  - type: 排名类型 (daily/weekly/monthly)
  - page: 页码
```
