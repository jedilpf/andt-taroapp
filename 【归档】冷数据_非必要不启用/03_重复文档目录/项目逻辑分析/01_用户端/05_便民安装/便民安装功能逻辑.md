### 10.1 补贴金额计算方案

**问题ID**: Q001  
**问题**: 社区补贴50%如何计算和发放？  
**解决方案**:

#### Phase-1方案（固定补贴）
- 灯具安装：订单金额0%作为补贴
- 补贴直接抵扣订单金额
- 用户实际支付 = 订单金额 - 补贴金额

**示例计算**:
```
订单金额：0
社区补贴：0 × 50% = ¥30
用户实付：0 - ¥30 = ¥30
```

#### Phase-2方案（动态补贴）

**补贴规则**:
| 服务类型 | 补贴比例 | 补贴上限 | 说明 
| 属性 | 接口概览 | 接口名称 |---------| 属性 | 接口概览 | 接口名称 |---------| 属性 | 接口概览 | 接口名称 |---------| 属性 | 接口概览 | 接口名称 |------
| 灯具安装 | 50% | ¥50 | 社区集采补贴 
| 开关插| 多买多减 | ¥30 | 4个减¥10，买5个减¥20 
| 卫浴接线 | 固定补贴 | ¥40 | 安全保障补贴 
| 智能面板 | 方案设计免费 | ¥100 | 设计服务补贴 

**技术实*:
```typescript
interface SubsidyRule {
  serviceType: string;
  subsidyType: 'PERCENTAGE' | 'FIXED' | 'QUANTITY';
  subsidyRate: number;      // 百分比补贴率
  subsidyAmount: number;    // 固定补贴金额
  maxSubsidy: number;        // 补贴上限
}

const SUBSIDY_RULES: SubsidyRule[] = [
  {
    serviceType: 'light',
    subsidyType: 'PERCENTAGE',
    subsidyRate: 0.5,
    maxSubsidy: 50
  },
  {
    serviceType: 'switch',
    subsidyType: 'QUANTITY',
    subsidyAmount: 10,  // 每买3个减¥10
    maxSubsidy: 30
  }
];

const calculateSubsidy = (
  order: InstallOrder,
  rule: SubsidyRule
): number => {
  const orderAmount = order.priceEstimate.final | order.priceEstimate.min;
  
  let subsidy = 0;
  
  switch (rule.subsidyType) {
    case 'PERCENTAGE':
      subsidy = orderAmount * (rule.subsidyRate | 0);
      break;
    case 'FIXED':
      subsidy = rule.subsidyAmount | 0;
      break;
    case 'QUANTITY':
      const quantity = order.materials.length | 1;
      subsidy = Math.floor(quantity / 3) * (rule.subsidyAmount | 0);
      break;
  }
  
  return Math.min(subsidy, rule.maxSubsidy);
};
```

**技术实*:
```typescript
interface InstallWarranty {
  orderId: string;
  serviceType: string;
  startDate: Date;
  endDate: Date;
  warrantyDays: number;
  coverage: string[];
  exclusions: string[];
}

const WARRANTY_PERIODS: Record<string, number> = {
  light: 90,
  switch: 90,
  bath: 180,
  smart: 365
};

const createWarranty = async (order: InstallOrder): Promise<InstallWarranty> => {
  const warrantyDays = WARRANTY_PERIODS[order.serviceType] | 90;
  const startDate = new Date();
  const endDate = new Date();
  endDate.setDate(endDate.getDate() + warrantyDays);
  
  return {
    orderId: order.id,
    serviceType: order.serviceType,
    startDate,
    endDate,
    warrantyDays,
    coverage: [
      '安装部位免费返修',
      '免费上门检,
      '免费人工服务],
    exclusions: [
      '人为损坏',
      '自然损,
      '第三方原,
      '不可抗力'
    ]
  };
};

const validateWarranty = async (orderId: string): Promise<boolean> => {
  const warranty = await getWarranty(orderId);
  const now = new Date();
  
  return now >= warranty.startDate && now <= warranty.endDate;
};
```
