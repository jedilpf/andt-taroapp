## 一、项目概述

### 1.1 项目简介

**安电通**是社区电力服务O2O平台，通过APP+小程序连接居民与专业电工，提供：
- **应急维修**：电路故障、跳闸、短路、插座损坏
- **便民安装**：灯具、开关、插座、智能家居
- **安全检测**：家庭电路安全检测、隐患排查
- **社区商城**：电器配件、智能家居产品

### 1.2 项目目标

| 阶段 | 周期 | 投入 | 核心目标 |
|-----|-----|-----|---------|
| 第1期 MVP验证 | M1-M6 (6个月) | 52万 | APP+小程序上线，验证需求 |
| 第2期 商业验证 | M7-M12 (6个月) | 68万 | 用户>3000，月收>5万 |
| 第3期 规模运营 | M13-M24 (12个月) | 170万 | 用户>15000，实现盈利 |

### 1.3 第1期产出

| 产出物 | 描述 | 交付时间 |
|-------|-----|---------|
| Android APP | React + Capacitor，生产级可用 | M6末 |
| 微信小程序 | Taro框架，核心功能完整 | M6末 |
| 后端服务 | 用户/订单/支付核心API | M6末 |

### 2.2 五大环节详解

#### 环节一：获客（Acquisition）

**目标**：低成本获取精准用户

| 获客渠道 | 方式 | 成本 | 预期转化 |
|---------|-----|-----|---------|
| **公益安检引流** | 免费上门电路检测 | 20元/人 | 30%注册转化 |
| **政府/社区合作** | 街道背书、社区活动 | 10元/人 | 40%注册转化 |
| **物业推荐** | 物业群推广、公告栏 | 15元/人 | 25%注册转化 |
| **口碑推荐** | 老用户推荐新用户 | 5元/人 | 50%注册转化 |
| **线上投放** | 朋友圈、本地公众号 | 50元/人 | 15%注册转化 |

**获客漏斗**：
```
社区活动曝光 1000人
    ↓ (扫码率 20%)
扫码访问 200人
    ↓ (注册率 50%)
注册用户 100人
    ↓ (首单率 30%)
付费用户 30人

获客成本：活动成本2000元 / 付费用户30人 = 67元/人
```

---

#### 环节二：转化（Conversion）

**目标**：将免费用户转化为付费用户

**转化路径**：
```
免费用户
    │
    ├── 路径A：公益安检 → 发现问题 → 付费整改
    │         (转化率 15-20%，客单价 100-300元)
    │
    ├── 路径B：应急维修 → 首次付费 → 复购
    │         (转化率 80%，客单价 150-300元)
    │
    └── 路径C：商城浏览 → 优惠券刺激 → 首次购买
              (转化率 5-10%，客单价 50-200元)
```

**转化触点设计**：

| 触点 | 触发条件 | 转化动作 | 预期效果 |
|-----|---------|---------|---------|
| 安检报告 | 发现安全隐患 | 推荐整改服务+优惠券 | 15%转化 |
| 服务完成 | 订单评价后 | 推送商城优惠券 | 10%复购 |
| 会员卡弹窗 | 第2次下单 | 会员卡年费优惠 | 5%会员转化 |
| 积分提醒 | 积分即将过期 | 商城兑换提醒 | 8%商城转化 |

**收入结构**：

```
┌─────────────────────────────────────────────────────┐
│                    收入来源                          │
├─────────────────────────────────────────────────────┤

│                                                     │
│  【商城收入】占比 35%                                │
│  ├── 电器配件销售：毛利 20-30%                      │
│  ├── 智能家居产品：毛利 15-25%                      │
│  └── 团购商品：毛利 10-15%                          │
│                                                     │
│  【会员收入】占比 10%                                │
│  ├── VIP年费：99元/年                              │
│  └── SVIP年费：299元/年                            │
│                                                     │
│  【广告收入】占比 10%                                │
│  ├── 品牌曝光广告                                   │
│  └── 商家推广位                                     │
│                                                     │
│  【其他收入】占比 5%                                 │
│  └── 数据服务、培训服务等                           │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**单位经济模型**：

```
用户生命周期价值（LTV）计算：

单用户年度贡献：
├── 服务订单：3单 × 150元 × 15%佣金 = 67.5元
├── 商城消费：500元 × 15%毛利 = 75元
├── 会员费：20%用户 × 99元 = 19.8元
├── 广告价值：10元
└── 合计：172.3元/年

用户生命周期（2年）：
LTV = 172.3 × 2 = 345元

获客成本（CAC）：
├── 公益安检获客：20元
├── 政府合作获客：10元
├── 口碑推荐获客：5元
└── 平均CAC：15-20元

LTV/CAC = 345 / 18 = 19倍（健康水平）
```

#### 环节五：裂变（Referral）

**目标**：低成本获客，指数级增长

**裂变机制**：

| 裂变方式 | 机制 | 奖励 | 预期效果 |
|---------|-----|-----|---------|
| **推荐有礼** | 邀请好友注册 | 双方各得20元券 | K因子 1.2 |
| **拼团优惠** | 3人拼团享折扣 | 发起人额外积分 | K因子 1.5 |
| **分享得积分** | 分享服务到朋友圈 | 获得50积分 | K因子 0.8 |
| **老带新** | 老用户推荐新用户 | 老用户得服务券 | K因子 1.3 |

**裂变漏斗**：
```
1000个老用户
    ↓ (分享率 20%)
200人分享
    ↓ (曝光率 ×10)
2000人看到
    ↓ (点击率 10%)
200人点击
    ↓ (注册率 50%)
100人注册
    ↓ (转化率 30%)
30人付费

裂变效果：每1000老用户 → 产生30个新付费用户
裂变成本：优惠券成本 × 30 = 约300元
新客成本：300 / 30 = 10元/人（远低于直接获客）
```
