一、OpenClaw 简介

OpenClaw 是一个开源的个人AI助手框架，拥有 35万+ Stars，支持：
✅ 22+ 消息渠道（微信、钉钉、Slack、Discord等）
✅ 技能扩展系统（SKILL.md）
✅ 多平台设备节点（macOS/iOS/Android）
✅ Gateway统一控制平面架构

二、架构设计

输入渠道 处理层 输出 ┌─────────┐ ┌─────────────┐ ┌─────────┐ │ 微信 │ │ OpenClaw │ │ GitHub │ │ 钉钉 │ ──→ Gateway ──→ │ Gateway │ ──→ 技能 ──→ │ Issues │ │ WebChat │ (WebSocket) │ + Skills │ │ 文档库 │ │ CLI │ │ │ │ 通知 │ └─────────┘ └─────────────┘ └─────────┘

三、核心技能设计

3.1 需求分析技能 (requirement-analyzer)

功能输入输出
需求解析用户需求描述澄清后的需求
任务分解需求理解任务列表+工时估算
风险识别技术方案风险点+应对建议
自动创建任务列表GitHub Issues

3.2 进度跟踪技能 (progress-tracker)

监控维度说明预警条件
任务完成率已完成/总数落后计划>10%
里程碑进度当前里程碑状态延迟>2天
资源利用率人员工作量分布负载>100%
延期风险可能延期的任务风险等级>高

3.3 文档生成技能 (doc-generator)

命令功能
/doc api从Controller代码生成API文档
/doc db从DDL语句生成数据库文档
/doc changelog从Git提交生成更新日志

四、安装部署

# 安装Node.js 24 nvm install 24 # 安装OpenClaw npm install -g openclaw@latest # 初始化配置 openclaw onboard --install-daemon # 配置模型 openclaw config set model "gpt-4o" # 发布技能 openclaw skill publish requirement-analyzer openclaw skill publish progress-tracker openclaw skill publish doc-generator
