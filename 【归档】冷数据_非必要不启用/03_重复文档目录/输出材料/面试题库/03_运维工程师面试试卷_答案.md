## 四、操作题答案

**16. Linux命令（10分）**

```
(1) 查找所有.log文件并删除：
    find / -name "*.log" -exec rm {} \;

(2) 查看端口8080被哪个进程占用：
    lsof -i:8080

(3) 解压一个tar.gz文件：
    tar -xzvf file.tar.gz

(4) 给脚本添加执行权限：
    chmod +x script.sh
```

每个命令2.5分

**17. Dockerfile（15分）**

```dockerfile
FROM openjdk:17-jdk-alpine

WORKDIR /app

COPY app.jar app.jar

EXPOSE 8080

ENTRYPOINT ["java", "-jar", "app.jar"]
```

评分标准：
- FROM正确（3分）
- WORKDIR正确（3分）
- COPY正确（3分）
- EXPOSE正确（3分）
- ENTRYPOINT正确（3分）
