三、简答题答案（每题8分）

11. 什么是Spring的依赖注入？它有什么优点？

评分要点：

  • 定义：对象不自己创建依赖，由容器注入（3分）
  • 优点：降低耦合、便于测试、便于维护（5分）

12. 如何优化MySQL慢查询？

评分要点：

  • 使用EXPLAIN分析执行计划（2分）
  • 检查索引使用情况（2分）
  • 避免SELECT *、优化WHERE条件（2分）
  • 合理使用索引、分页优化（2分）

13. Redis有哪些使用场景？请列举至少3个。

评分要点：

  • 缓存（3分）
  • 会话存储（2分）
  • 分布式锁（2分）
  • 计数器/排行榜（1分）

14. 什么是JWT？它的工作原理是什么？

评分要点：

  • 定义：JSON Web Token，用于认证的令牌（3分）
  • 组成：Header.Payload.Signature（2分）
  • 工作原理：登录生成token、请求携带token、服务端验证（3分）

15. 如何保证接口的幂等性？

评分要点：

  • 使用唯一标识（订单号、token）（3分）
  • 数据库唯一索引（2分）
  • Redis分布式锁（2分）
  • 状态机校验（1分）

四、编程题答案

16. SQL查询题（10分）

SELECT u.id, u.name, COUNT(o.id) as order_count, IFNULL(SUM(o.amount), 0) as total_amount FROM user u LEFT JOIN `order` o ON u.id = o.user_id GROUP BY u.id, u.name;

评分标准：LEFT JOIN正确（3分）、COUNT和SUM函数正确（4分）、GROUP BY正确（3分）

17. 单例模式（15分）

public class Singleton { private static volatile Singleton instance; private Singleton() {} public static Singleton getInstance() { if (instance == null) { synchronized (Singleton.class) { if (instance == null) { instance = new Singleton(); } } } return instance; } }

评分标准：私有构造函数（3分）、volatile关键字（4分）、双重检查锁（5分）、线程安全（3分）
