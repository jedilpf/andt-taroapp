## 三、简答题答案（每题8分）

**11. 什么是React的虚拟DOM？它有什么优势？**

评分要点：
- 虚拟DOM是JavaScript对象表示的DOM结构（2分）
- 在内存中操作，不直接操作真实DOM（2分）
- 优势：批量更新、性能优化、跨平台（2分）
- 通过diff算法计算最小变更（2分）

**12. useEffect的依赖数组有几种用法？分别是什么效果？**

评分要点：
- 不传数组：每次渲染都执行（2分）
- 空数组[]：只在挂载时执行一次（2分）
- [value]：value变化时执行（2分）
- 返回清理函数：组件卸载或下次执行前调用（2分）

**13. 什么是防抖（debounce）？它有什么应用场景？**

评分要点：
- 定义：在事件触发后等待一段时间再执行，期间再次触发则重新计时（3分）
- 作用：减少函数执行频率（2分）
- 应用场景：搜索框输入、窗口resize、表单验证（3分）

**14. 简述Tailwind CSS的优缺点。**

评分要点：
- 优点：快速开发、无需写CSS文件、响应式简单、自动清除未使用样式（4分）
- 缺点：类名较长、HTML臃肿、学习成本（4分）

**15. 什么是响应式设计？如何实现移动端适配？**

评分要点：
- 定义：让网页适配不同屏幕尺寸（2分）
- 实现：媒体查询、rem/vw单位、Flex布局、 viewport设置（6分）

## 四、编程题答案

**16. 计数器组件（10分）**

```jsx
function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>当前计数: {count}</p>
      <button onClick={() => setCount(count + 1)}>+1</button>
      <button onClick={() => setCount(count - 1)}>-1</button>
      <button onClick={() => setCount(0)}>重置</button>
    </div>
  );
}
```

评分标准：
- useState正确使用（3分）
- 显示当前计数（2分）
- 三个按钮功能正确（5分）

**17. 防抖函数（15分）**

```javascript
function debounce(fn, delay) {
  let timer = null;
  return function(...args) {
    clearTimeout(timer);
    timer = setTimeout(() => {
      fn.apply(this, args);
    }, delay);
  };
}
```

评分标准：
- 使用闭包保存timer（5分）
- clearTimeout清除定时器（5分）
- setTimeout延迟执行（5分）
