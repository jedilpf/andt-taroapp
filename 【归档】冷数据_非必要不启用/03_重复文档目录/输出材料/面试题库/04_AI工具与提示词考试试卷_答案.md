## 三、简答题答案（每题8分）

**11. 什么是RAG（检索增强生成）？它有什么价值？**

评分要点：
- 定义：先从知识库检索相关信息，再让AI基于检索结果生成回答（4分）
- 价值：使用企业内部知识、减少幻觉、知识可更新（4分）

**12. 如何让AI扮演特定角色回答问题？请举例说明。**

评分要点：
- 在提示词开头指定角色（4分）
- 示例："你是一位有10年经验的Java开发工程师..."（4分）

**13. 当AI生成的代码有bug时，你会如何处理？**

评分要点：
- 先理解bug原因（2分）
- 把错误信息发给AI（2分）
- 让AI分析并修复（2分）
- 验证修复方案（2分）

**14. 如何通过提示词让AI输出JSON格式的数据？**

评分要点：
- 明确要求输出JSON格式（4分）
- 提供示例格式（4分）

**15. 使用AI工具时，如何保护敏感信息？**

评分要点：
- 不输入密码、密钥等敏感信息（3分）
- 使用匿名数据测试（3分）
- 了解AI的数据使用政策（2分）

## 四、提示词编写题答案

**16. 登录表单组件生成提示词（10分）**

参考答案：

```
请帮我生成一个React登录表单组件，要求：

功能要求：
1. 包含手机号输入框（验证11位数字格式）
2. 包含验证码输入框（6位数字）
3. 包含"获取验证码"按钮
4. 包含"登录"按钮

样式要求：
1. 使用Tailwind CSS
2. 表单居中显示
3. 输入框有边框和圆角
4. 按钮为橙色背景、白色文字

输出要求：
1. TypeScript代码
2. 包含类型定义
```

评分标准：
- 功能要求清晰（4分）
- 样式要求具体（3分）
- 输出格式明确（3分）

**17. SQL优化提示词（15分）**

参考答案：

```
请帮我分析以下SQL的性能问题并给出优化方案：

SQL语句：
[粘贴SQL]

表结构：
[粘贴表结构]

请分析：
1. 是否使用了索引？如果没有，建议添加什么索引？
2. 是否存在全表扫描？
3. 是否有不必要的子查询？
4. 请给出优化后的SQL语句
5. 预估优化后的性能提升

数据库版本：MySQL 8.0
```

评分标准：
- 提供待分析内容（4分）
- 明确分析维度（6分）
- 指定输出要求（5分）
