# 安电通APP图标设计方案

> **版本**: V1.0  
> **更新日期**: 2026年3月  
> **设计说明**: 基于项目定位和品牌调性的图标设计方案

---

## 一、设计理念

### 1.1 品牌定位

**安电通** - 家庭电路服务的滴滴
- **核心价值**: 安全、专业、便捷
- **目标用户**: 家庭用户 + 专业电工
- **服务场景**: 电路急修、深度体检、上门安装

### 1.2 设计关键词

```
安全 🔒    专业 ⚡    便捷 📱
电路 ⚡    服务 👷    信任 🤝
```

## 二、图标设计方案

### 方案一：闪电+盾牌（推荐）

#### 设计说明

结合"电路"和"安全"两个核心元素，传达专业电工服务的安全感。

#### 视觉元素

```
┌─────────────────┐
│    ⚡ 闪电       │  ← 电路/电工元素
│   ╱   ╲         │
│  ╱  🛡️  ╲       │  ← 盾牌/安全元素
│ ╱_________╲     │
│   安电通        │  ← 品牌名称
└─────────────────┘
```

#### 色彩方案

| 元素 | 颜色 | 色值 | 说明 |
|------|------|------|------|
| 闪电 | 电光蓝 | #00D4FF | 科技感、电路 |
| 盾牌 | 深蓝 | #1E3A8A | 专业、信任 |
| 背景 | 渐变 | #1E3A8A → #00D4FF | 从深蓝到电光蓝 |

#### 适用场景

- ✅ 用户端APP图标
- ✅ 电工端小程序图标
- ✅ 品牌Logo

## 三、图标规格

### 3.1 尺寸规格

| 平台 | 尺寸 | 格式 | 说明 |
|------|------|------|------|
| iOS | 1024×1024px | PNG | App Store |
| Android | 512×512px | PNG | Google Play |
| 小程序 | 81×81px | PNG | 微信/支付宝 |
| 网站 | 32×32px - 192×192px | PNG/ICO | Favicon |

### 3.2 设计规范

#### 安全区域

```
┌─────────────────────────────┐
│                             │
│    ┌───────────────────┐    │
│    │                   │    │  ← 安全区域
│    │    图标内容        │    │     (占80%)
│    │                   │    │
│    └───────────────────┘    │
│                             │
└─────────────────────────────┘
```

#### 圆角规范

| 平台 | 圆角 | 说明 |
|------|------|------|
| iOS | 180px (1024px图标) | 苹果标准圆角 |
| Android | 自适应 | 支持自适应图标 |
| 小程序 | 无圆角 | 正方形 |

## 七、设计检查清单

### 7.1 视觉检查

- [ ] 图标在小尺寸下依然清晰可辨
- [ ] 色彩对比度符合WCAG标准
- [ ] 圆角处理符合平台规范
- [ ] 安全区域内无重要元素被裁切

### 7.2 技术检查

- [ ] 所有尺寸图标已导出
- [ ] 文件大小符合平台要求
- [ ] 透明背景处理正确
- [ ] 矢量文件可正常编辑

### 7.3 品牌检查

- [ ] 与品牌调性一致
- [ ] 与竞品有明显区分
- [ ] 用户易于识别和记忆
- [ ] 适用于各种应用场景
