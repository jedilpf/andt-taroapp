## 快速开始

### 前端开发

```bash
# 用户端
cd andiantong-app
npm install
npm run dev

# 电工端
cd andiantong-enterprise
npm install
npm run dev:h5        # H5开发
npm run dev:weapp     # 微信小程序
```

### 后端开发

```bash
# 启动Nacos
cd nacos/bin
startup.cmd -m standalone

# 创建数据库
mysql -u root -p
CREATE DATABASE andiantong_user;
CREATE DATABASE andiantong_order;
CREATE DATABASE andiantong_task;

# 启动服务
cd gateway && mvn spring-boot:run
cd user-service && mvn spring-boot:run
```
