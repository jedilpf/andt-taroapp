## 二、市场分析

### 2.1 行业背景与痛点

#### 四大核心痛点

| 痛点 | 现状描述 | 解决方案 |
|------|---------|---------|
| 找电工难、慢、贵 | 缺乏透明定价和服务保障 | 实时地图 + 标准化定价 + 平台认证 |
| 家庭用电安全隐患高发 | 电路老化、违规用电问题普遍 | 首次免费检测 + AI诊断 |
| 服务后缺乏跟踪 | 一次服务结束无持续保障 | 90天质保 + 保险兜底 |
| 社区服务分散 | 居民需求无法一站式满足 | 商城闭环 + 便民服务集成 |

#### 市场数据支撑

| 数据 | 说明 |
|------|------|
| 3.3亿 | 60岁以上老年人群体，存在用电安全需求 |
| 老龄化趋势 | 老小区用电安全隐患加剧 |
| 家政市场 | 持续扩大，电工服务占比小但潜力大 |

### 4.2 三大盈利板块

根据3.25会议讨论，商业模式分为**三大盈利板块**：

| 板块 | 说明 | 阶段 |
|------|------|------|
| **服务佣金** | 交易抽成（前期20-25%，后期调整） | 核心收入 |
| **材料差价** | 平台代购材料，批量采购优惠 | 稳定收入 |
| **会员订阅** | 月度免费检测次数等权益 | 增值收入 |

**积分体系**：积分来源于商城消费，**可抵物业费**（这是用户留存的核心）。

### 4.3 定价策略

| 策略 | 说明 |
|------|------|
| 途虎模式 | 检查免费 → 问题修复+材料打包 → **一次性收费** |
| 定价示例 | 199/299元包含：检测+维修+后续服务 |

| 签到激励 + 连续打卡奖励 | 提升日活和留存 | 日常打开 |
| 便民功能集成 | 宠物医院、便民电话、社区资讯 | 一站式服务 |
| 强时效推送 | 诈骗预警、火灾通报、活动通知 | 实用价值 |
| 实用消耗品推广 | 插座、灯泡等切入 | 培养使用习惯 |

### 4.5 电工端激励

| 阶段 | 重点 | 方式 |
|------|------|------|
| 前期 | 社区街道 + 物业合作 | 地推团队 + 物业联动 |
| 中期 | 微信生态裂变 | 邀请有礼、拼团 |
| 后期 | 口碑传播为主 | 用户自发推荐 |
